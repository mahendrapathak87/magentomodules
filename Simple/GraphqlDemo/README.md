# Module Title

Simple graphQl module , demonstrates how to create request with graphQl api and use it 

## Getting Started

http://mymagento.com/graphql

where http://mymagento.com is your domain name

### Prerequisites

Magento 2.3 installation with working graphQl
http://mymagento.com/graphql

  }
}
